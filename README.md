📚 Library Management System

A desktop-based Library Management System developed using Python, Tkinter, and MySQL. This application helps manage books, track issued/returned books, and maintain library records efficiently through a user-friendly GUI.

🚀 Features
Add, update, and delete books
Book search functionality
Issue and return books
Manage library records using MySQL database
Interactive GUI using Tkinter
Real-time data updates


🛠️ Tech Stack
Frontend (GUI): Tkinter (Python)
Backend: Python
Database: MySQL
Connector: mysql-connector-python


📂 Project Structure
library-management-system/
│── main.ipynb / main.py
│── database.sql
│── README.md
│── requirements.txt
